Sahayak AI is built on one observation every welfare tool has gotten wrong — people don't know what they don't know. They can't search for something they've never heard of. So instead of a search bar, Sahayak opens with a conversation. It asks users to describe their situation in plain language, builds a silent profile from that dialogue, and generates a Support Gap Report — everything they qualify for that they're not currently receiving, ranked by urgency, with exact steps to claim it.
